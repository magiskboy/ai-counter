# Sandbox seed data

Mirrors what `./install.sh` copies to `~/.sandbox-ai-counter`. Does **not** include sample projects — add your own git repos under `projects/` and list them in `ai-counter/config.yaml`.

```
sandbox/
├── ai-counter/
│   ├── config.yaml
│   └── prompts/daily.yaml
├── .cursor/{mcp.json,cli-config.json}
└── projects/          # empty; clone your repos here after install
```
